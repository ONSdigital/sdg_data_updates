calculate_valid_rates_per_1000 {SDGupdater}R Documentation

Rates per 1000

Description

Calculates the rate per 1000, when the numerator is greater than 3.

Usage

calculate_valid_rates_per_1000(numerator, denominator, decimal_places)

Arguments

numerator, denominator, decimal_places

Numeric vector.

Value

Numeric vector.

Examples

calculate_valid_rates_per_1000(13, 11000, 2)
calculate_valid_rates_per_1000(3, 11000, 2)
