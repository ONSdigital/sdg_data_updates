get_modal_values {SDGupdater}R Documentation

Get modal values

Description

Finds which value occurs most frequently. If there is more than one mode, all will be returned (i.e. this works for multimodal data)

Usage

get_modal_values(values)

Arguments

values

Numeric or character vector.

Value

Vector of same type as values.

See Also

From thread: https://stackoverflow.com/questions/2547402/how-to-find-the-statistical-mode

Examples

get_modal_values(c(1,2,3,1,3))
get_modal_values(c(1,1,2,3))
get_modal_values(c("a", "a", "b"))
